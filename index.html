文件处理命令

ls

命令 功能
ls 查看该级文件
ls -a 查看该级隐藏文件
ls / 查看该级所有文件
ls -l 查看长文件信息
ls -lh 大小用k、M、G显示
ls -d 查看当前目录信息
ls -i 查看文件id
ls xx+tab 补齐文件开头为xx的文件
