字符串

字符串hash

​ 字符串hash是指将一个字符串S映射为一个整数，使得该整数可以尽可能唯一地代表S。如果转换的整数相同，则代表的S也相同。

​ 下面为hash值的递推公式，其中$str[i]$表示字符串的第$i$位，index函数将A-Z分别转换为数值0-25，$H[i]$表示字符串的前$i$个字符串的hash值。

$$H[i] = H[i - 1] * 26 + index(str[i])$$

​ 以上公式，虽然字符串与整数是一一对应的，但由于没有适当处理，因此当长度较长，产生的整数过大，没法保存。为了应对，放弃“唯一性”，进行取模运算。

$$H[i] = (H[i - 1] * 26 + index(str[i])) % mod$$

滑动窗口

滑动窗口的本质是一个双指针，它并不局限于字符串类问题，主要用于解最优解问题。

步骤

  1. 设置l = 0和r = 0两个指针，[l, r]的区间看作一个窗口。
  2. 通过++r逐步将指针右移，查看是否有匹配原题的答案（例如字符串匹配，或者是加和相等），直到有匹配的答案时停止右移。
  3. 然后通过++l的方式缩短区间[l, r]，直到该区间内的答案不再匹配。更新当前的ans（可以是窗口大小）
  4. 重复2、3步，直到l == n - 1，此时的ans即为结果。

本质上来看，第2步在寻找可行解，第三步在优化可行解。

trie树

定义

​ 又称单词查找树，Trie树，是一种树形结构，是一种哈希树的变种。典型应用是用于统计，排序和保存大量的字符串（但不仅限于字符串），所以经常被搜索引擎系统用于文本词频统计。它的优点是：利用字符串的公共前缀来减少查询时间，最大限度地减少无谓的字符串比较，查询效率比哈希树高。

定义

位运算其实就是直接对内存中的二进制数据进行操作，因此速度非常快。

基础位操作符

符号 描述 运算规则
& 与 两个位都为1时，结果才为1
` ` 或
^ 异或 两个位相同为0，相异为1
~ 取反 0变1，1变0
<< 左移 各二进位全部左移若干位，高位丢弃，低位补0
>> 右移 各二进位全部右移若干位，对无符号数，高位补0，有符号数，各编译器处理方法不一样，有的补符号位（算术右移），有的补0（逻辑右移）

注意：

  • 只有~是单目运算符
  • 位操作只作用于int
  • VC6.0和VS2008的编译器会将>>按算术右移处理
  • 位运算符的运算级比较低
  • 除以上还有复合操作符，&=、|=、 ^=、<<=、>>=。
