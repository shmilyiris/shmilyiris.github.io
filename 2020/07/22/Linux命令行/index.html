文件处理命令

绝对路径与相对路径

ls

命令 功能
ls 查看该级文件
ls -a 查看该级隐藏文件
ls / 查看该级所有文件
ls -l 查看长文件信息
ls -lh 大小用k、M、G显示
ls -d 查看当前目录信息
ls -i 查看文件id
ls xx+tab 补齐文件开头为xx的文件

pwd

print working directory 输出当前路径

cd

命令 功能
cd /home/test 访问xx目录
cd .. 访问上一级
cd / 访问根目录
cd ~ 访问主目录(home)

mkdir

命令 功能
mkdir test 新建test文件夹
mkdir -p /test/123 递归创建目录
mkdir t1 t2 t3 同时创建多个目录

mv

移动文件或重命名

命令 功能
mv file_1 file_2 将file_1重命名为file_2
mv file /dir 将file移动到目录dir中
mv /dir1 /dir2 将目录dir1移动到dir2
mv /dir1/* 将dir1下的文件移动到当前目录

rm

命令 功能
rmdir test 删除一个空目录
rm -rf abc 将当前目录中所有内容强制删除

cp

命令 功能
cp-rf e1 e2 复制文件或目录
move hi hello 重命名文件或目录

归档压缩

命令 功能
tar -cvf f.tar f1 f2 f3 将文件归档成档案包
tar -xvf f.tar 还原档案包到当前路径
tar -xvf f.tar -C xxx/ 还原档案包到指定路径
tar -zcvf f.tar.gz f1 f2 f3 归档并压缩
tar -zxvf f.tar.gz 解压缩到当前路径

模式转换

  • 命令模式
  • 输入模式
  • 底线命令模式

  • i切换到输入模式，以输入字符

  • :切换到底线命令模式，以在最底一行输入命令。

常用命令

光标移动

操作 说明
[Ctrl] + [f] [Page Down]
[Ctrl] + [b] [Page Up]
[Ctrl] + [d] 屏幕向下移动半页
[Ctrl] + [u] 屏幕向上移动半页
+ 光标移动到非空格符的下一行
- 光标移动到非空格符的上一行
n [Space] 后退几格
n [Enter] 后退几行
gg 移动到第一行
G 移动到最后一行
[0]/[Home] 移动到该行头部
[$]/[End] 移动到该行尾部

搜索替换

操作 说明
/word 从光标向下找名为word的字符串
?word 从光标向上找名为word的字符串
n n代表重复上一个搜索的动作
N N代表反向重复上一个动作

删除复制与粘贴

操作 说明
dd 删除游标所在的一整行
yy 复制游标所在的一整行
ndd 删除光标所在的向下n行
d1G 删除光标所在到第一行的所有数据
dG 删除光标所在到最后一行的所有数据
d$ 删除光标到该行末尾的数据
d0 删除光标到该行开头的数据
u 复原前一个动作
[Ctrl] + r 重做上一个动作
