STL整理

发表于 2020-07-29 更新于 2021-03-01 分类于 应试 阅读次数： Valine：

是一个标准模板库的使用字典。

vector

头文件

#include <vector>

访问方式

单个访问

  1. 下标访问
  2. begin()函数(*(v.begin + i)) 只有string和vector支持这种访问方法

遍历

  1. for循环利用下标i
  2. for循环利用迭代器
    • for(vector<int>::iterator it = v.begin(); it != v.end(); ++it)
    • for(auto temp:v)

常用函数

  • push_back(x) —$O(1)$
  • pop_back() —$O(1)$
  • size() —$O(1)$
  • clear() —$O(n)$
  • insert(it, x) —$O(n)$
  • erase(it)/ erase(first, last) —$O(n)$

常见用途

  1. 存储数据
  2. 充当数组
  3. 邻接表存储图

set

内部有序且不含重复元素的容器。

头文件

#include <set>

访问方式

只能通过迭代器访问

常用函数

  1. insert(x) —$O(logn)$

  2. find(value) —$O(logn) $

    返回的是对应值为value的迭代器

  3. erase()

    • 通过迭代器删除单个元素 st.erase(it) —$O(1)$

      配合find()使用场景：st.erase(st.find(100))

    • 通过值删除单个元素st.erase(value) —$O(logn)$

    • 通过迭代器删除区间元素 st.erase(first, last) — $O(last - first)$

  4. size() —$O(1)$

  5. clear() —$O(n)$

常见用途

  1. 自动去重并按升序排列
  2. 扩展容器unordered_set，只去重，不排序。

string

头文件

#include <string>

访问方式

  1. 下标访问
  2. 迭代器访问

常用函数

  1. +=

  2. >、<、==(比较字典序)

  3. length() / size() —$O(1)$

  4. insert() —$O(n)$

    • str.insert(pos, string) 向str[3]的位置插入字符串string

    • str.insert(it, it2, it3)向str字符串中插入另一字符串的首尾迭代器it2和it3之间

  5. erase() —$O(n)$

    • erase(it)
    • erase(first, last)
    • erase(pos, length)

  6. clear() —$O(1)$

  7. substr(pos, len) —$O(len)$

  8. find() —$O(nm)$

    • str.find(str2)

      当str2是str的子串，返回其在str中第一次出现的位置。否则返回npos。

    • str.find(str2, pos)

      以pos（int）位为起点，开始向后匹配。

  9. string::npos 当作find()函数失配的返回值

  10. replace() —$O(str.length())$

    • str.replace(pos, len, str2)

      把str从pos位开始，长度为len的子串替换为str2

    • str.replace(it1, it2, str2)

map

map可以建立键值对的映射，且会以键从小到大的顺序自动排序。

头文件

#include <map>

访问方式

  1. 下标访问
  2. 迭代器访问
    • key：it->first
    • val：it->second

常用函数

  1. find(key) —$O(logn)$
  2. erase()
    • erase(it) —$O(1)$
    • erase(key) —$O(logn)$
    • erase(first, last) —$O(last - first)$
  3. size() —$O(1)$
  4. clear() —$O(1)$

常见用途

  1. 建立字符(字符串)到整数的映射
  2. 判断大整数或者其他数据是否存在
  3. 字符串映射
  4. unordered_map建立哈希表

常用函数

priority_queue

简介

优先队列，底层用堆进行实现。队列中，队首元素一定是当前队列中优先级最高的元素。

头文件

#include <queue>

访问方式

q.top()访问队首元素

常用函数

  1. push() —$O(logn)$
  2. top() —$O(1)$
  3. pop() —$O(logn)$
  4. empty() —$O(1)$
  5. size() —$O(1)$

优先级设置

基本数据类型设置

priority_queue<int> q

优先级按照元素从大到小降序排列

priority_queue<int, vector<int>, greater<int> > q;

优先级按照元素从小到大降序排列

结构体的优先级设置

以水果为例，根据水果的名称和价格建立一个结构体。

现在希望按照其价格从高到低获取水果名称，需要重载运算符，重载运算符函数加入到结构体的定义中

此时，可以直接定义fruit类型的优先队列priority_queue<fruit> q;，其内部以价格高的水果为高优先级。

如想以低价格为优先级高，只需将return语句中的<改成>即可。

另外，如果想将重载写在结构体之外，声明方式和队列的定义方式也需要发生一定的改变。

其他的STL容器也可以用类似的方法来定义优先级。

lower_bound() & upper_bound()

这两个函数需要用在一个有序数组或容器中。

复杂度 —$O(log(last-first))$

lower_bound(first, last, val)用来寻找在数组或容器的$[first, last)$范围内第一个值大于等于val的元素的位置。

  • 如果是数组，则返回该位置指针。
  • 如果是容器，则返回该位置迭代器。

upperbound(frist, last, val)用于寻找在数组或容器的$[first, last)$范围内第一个大于val的元素的位置。

  • 如果是数组，则返回该位置指针。
  • 如果是容器，则返回该位置迭代器。

如果数组或容器中，没有需要寻找的元素，则返回可以插入该元素位置的指针或迭代器。

如果只想获取其下标，则将返回值减去首地址/首迭代器。
