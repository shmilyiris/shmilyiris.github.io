BS4库

BS4库主要用于HTML页面的解析。是解析、遍历、维护“标签树”的功能库。

soup便承载了解析后的标签树。

基本元素 说明
Tag 标签，最基本的信息组织单元，分别用<>和</>标明开头和结尾
Name 标签的名字，格式：<tag>.name
Attributes 标签的属性，字典形式组织，格式：<tag>.attrs
NavigableString 标签内非属性字符串，格式：<tag>.string
Comment 标签内字符串的注释部分，一种特殊的Comment类型
