动态规划

发表于 2020-07-04 更新于 2020-07-18 分类于 算法题目 阅读次数：

20世纪50年代初美国数学家R.E.Bellman等人在研究多阶段决策过程的优化问题时，提出了著名的最优化原理(principle of optimality)，把多阶段过程转化为一系列单阶段问题，创立了解决这类过程优化问题的新方法——动态规划。

动态规划组成部分一：确定状态

两个意识
  • 最后一步
  • 子问题

状态由
$$ f(x)=最少用多少枚硬币拼出27 $$
转换为
$$f(x)=最少用多少枚硬币拼出27-a_{k}$$
归纳为
$$状态f(x)=最少用多少枚硬币拼出x$$

根据最后一步分部考虑：

$$f(27) = min{f(27-2)+1, f(27-5)+1, f(27-7)+1}$$

$$f[X] = min{f[X-2]+1, f[X-5]+1, f[X-7]+1}$$

动态规划组成部分三：初始条件和边界情况

动态规划组成部分概括

  1. 确定状态
    • 最后一步（最优策略中使用的最后一枚硬币$a_k$）
    • 化成子问题（最少的硬币拼出来更小的面值$27-a_k$）
  2. 转移方程
    • $f[X] = min{f[X-2]+1, f[X-5]+1, f[X-7]+1}$
  3. 初始条件和边界情况
    • f[0] = 0，如果不能拼出Y，f[Y]=inf
  4. 计算顺序
    • 正序

利用动态规划解实例2

  1. 确定状态
    • 最后一步：到达(m-1,n-1)前的方法是以M种方法到(m-2,n-1)加上以N种方法到(m-1,n-2)
    • 化成子问题：有多少种方法到达(m-2,n-1)或(m-1,n-2)
    • 状态：设$f[i][j]$为从左上角有多少种路径走到$(i,j)$
  2. 转移方程
    • $f[i][j] = f[i-1][j] + f[i][j-1]$
  3. 初始条件和边界情况
    • 初始条件：$f[0][0]=1$
    • 边界情况：$i=j=0$时，$f[i][j]=1$
  4. 计算顺序
    • 由左到右，逐行计算

利用动态规划解实例3

  1. 确定状态
    • 最后一步：到达n-1之前已经到达i
    • 化成子问题：跳到n-1 -> 跳到i
    • 状态：设$f[j]$为青蛙能不能跳到石头j
  2. 转移方程
    • $f[j]=OR_{0<=i<j}(f[i] AND i +a[i]>=j)$

    • 初始条件：$f[0]=true$
  4. 计算顺序
    • 从小到大

利用动态规划解实例4

  1. 确定状态
    用$f[i]$表示第i天结束后的累计最大收益，状态分为三种：

    • 持股状态：目前持有一种股票，累计最大收益为$f[i][0]$
    • 冷冻期状态：该天结束后处于冷冻期，对应的累计最大收益为$f[i][1]$
    • 双非状态：既不持有股票也不处于冷冻期状态，对应$f[i][2]$

  2. 转移方程
    状态转移：
    $$f[i][0] = max{f[i-1][0], f[i-1][2]-prices[i]}$$
    $$f[i][1] = f[i-1][0] + prices[i]$$
    $$f[i][2] = max{f[i-1][2], f[i-1][1]}$$
    结果：
    $$result = max{f[n-1][1],f[n-1][2]}$$

  3. 初始条件和边界情况
    $$f[0][0] = -prices[0]$$
    $$f[0][1] = 0$$
    $$f[0][2] = 0$$

利用动态规划解实例5

  1. 确定状态：$f[i][j]=s_1的前i个和s_2的前j个能否交错成s3的前i+j个$
  2. 转移方程：
    $$f[i][j] = [f[i-1][j] \ and \ s_1[i-1]=s_3[p]] or [f[i][j-1] \ and \ s_2[j-1]=s_3[p]]$$
    其中$p=i+j-1$
  3. 初始条件和边界情况：
    • 初始条件：$f[0][0]=true$
    • 边界条件：注意数组下标是否溢出
  4. 计算顺序：从小到大
